OPTION 1: USE ON VERCEL (Easiest)
──────────────────────────────────
1. Create Vercel account: vercel.com
2. Push dps-crm-simple.jsx + styles.css to GitHub
3. Connect GitHub to Vercel
4. Deploy in 1 click
5. Get live URL in seconds!
6. Share with your team


OPTION 2: RUN LOCALLY
──────────────────────
1. Have Node.js installed
2. Create: npx create-react-app dps-crm
3. Replace App.jsx with dps-crm-simple.jsx
4. Replace App.css with styles.css
5. Run: npm start
6. Visit: http://localhost:3000
